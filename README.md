# 智能办公文件分类与处理系统

> **AI-Powered Office Document Classification & Processing System**

## 项目概述

本项目设计并实现了一套**智能化的办公文件分类与处理系统**，基于大语言模型（LLM）实现对多源异构文件的自动识别、分类、评价和优先级处理。该系统能够自动处理txt、pdf、docx、jpg、png、xlsx等六种常见办公格式文件，并做出智能归档决策。

### 核心价值

| 维度 | 描述 |
|------|------|
| **效率提升** | 相比人工分类，处理速度提升数十倍 |
| **一致性** | 统一的分类标准，避免人工判断的主观差异 |
| **可扩展性** | 支持自定义分类体系和新的文件类型 |
| **智能决策** | 自动识别高优先级文件，优化人工复核资源 |

## 项目背景

企业日常运营中积累了大量多源异构文件，这些文件具有以下特点：

1. **来源多样**：来自不同部门、业务系统、人员
2. **格式各异**：包括文本、表格、图片、PDF等多种形式
3. **结构不一**：部分为结构化数据，部分为非结构化文档
4. **内容交叉**：同一文件可能涉及多个业务领域

传统的人工分类方式存在效率低、标准不统一、难以应对大规模数据等问题。本项目利用AI技术实现文件的智能化处理。

## 核心创新亮点

### 亮点一：模型无关的工厂模式设计

```python
class ModelFactory:
    _providers = {
        'ollama': OllamaModel,
        'lm_studio': LMStudioModel,
        'vllm': VLLMModel,
        'deepseek': DeepseekModel,
        'qwen': QwenModel,
        # ... 支持 11+ 种模型
    }
```

**优势**：
- **灵活切换**：通过配置文件即可切换不同的LLM后端
- **本地优先**：支持完全本地部署，保护数据隐私
- **成本可控**：可根据需求选择免费或付费模型

### 亮点四：纯规则评价体系

针对**无监督分类**场景，设计了不同于传统有监督的评价指标：

| 指标 | 计算方法 | 优势 |
|------|----------|------|
| **纯度 (Purity)** | 类别内高频词占比 | 反映类内一致性 |
| **轮廓系数** | 类内紧密度 vs 类间分离度 | 国际标准聚类评估 |
| **可解释性** | 有效分类比例 | 确保分类有意义 |
| **迁移适用性** | 分类多样性与覆盖率的平衡 | 评估泛化能力 |

### 亮点五：三维度优先级决策

```python
# 权重设计：0.4 + 0.3 + 0.3
priority = (
    0.4 * 紧急程度(urgency) +    # 时效要求较高
    0.3 * 错分风险(risk) +        # 主题归属不明确
    0.3 * 复核必要性(review)     # 涉及资金分配
)
```

**优势**：
- **纯规则实现**：无需AI调用，高效且可解释
- **符合B题要求**：明确考虑三类重点文件
- **可调权重**：可根据业务需求灵活调整
